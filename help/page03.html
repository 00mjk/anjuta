Creating a new file.

To create a new file just select the menu file/new or click the new file icon in the main tool-bar. A new page will appear on the editor (or a new window if the parent document is windowed). This new document will be named "newfile#1", "newfile#2", "newfile#3" ... and so on. You must save new documents as soon as possible to avoid any data loss due to emergency exit or application crash.
