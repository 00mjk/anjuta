Setting compiler options:

You can customize the the way the compiler will compile your file or Project. Remember, each project will have its own set of compiler settings. That is: what you set for a project is not available to other projects. They will have their own settings. When no project is opened, the settings are saved separately and are meant for compiling and building single files only. These are all transparent to the user. Anjuta handles them automatically.

The followings are the settings you can perform on the compiler:

  1. Specifying Include paths.
  2. Specifying Library paths.
  3. Specifying Libraries to link with.
  4. Defining macros.
  5. Specifying supports for your program.
  6. Compiler warnings.
  7. Code optimizations.
  8. Other compiler settings.
