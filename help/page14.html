Defining macros:

Activate the menu item 'Settings/Compiler options' and Select the 'Defines' page in the settings dialog box.
 Enter all the Defines one by one and press the Add button for each entry.

Remember, you have to only enter the macro name and no other else (as shown in figure).

To remove an entry, select the entry in the list and click Remove.

To edit an entry, select the entry, modify it in the entry box and click Update.

To clear all the entries, click Clear.
