Compiler warnings:

Activate the menu item 'Settings/Compiler options' and Select the 'Warnings' page in the settings dialog box.

Check the warnings you want to output in the message box when 
you compile or build your file or project.

Checking 'No warnings' will disable all warnings.

Similarly, checking All warnings will enable all warnings.
